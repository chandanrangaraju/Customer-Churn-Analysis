## Customer Churn Analysis

### Objective
To analyze customer behavior and identify factors contributing to customer churn.

### Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

### Workflow
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Model training and evaluation

### Key Insights
- Customers with shorter tenure are more likely to churn
- Contract type has a strong impact on churn behavior
- Higher monthly charges correlate with increased churn

### Conclusion
This project demonstrates how data analysis and machine learning
can help businesses understand churn patterns and improve retention strategies.
